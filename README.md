# 🏥 Hospital Management System

A Hospital Management System developed using **Java** and **DBMS** to efficiently manage hospital operations.

## 🚀 Features
- Patient Registration & Records
- Doctor Management
- Appointment Booking System
- Database Management using SQL
- User-friendly Interface

## 🛠️ Technologies Used
- Java
- MySQL / DBMS
- JDBC

## 📂 Project Structure
- src/ → Java source code
- database/ → SQL files
- docs/ → Documentation

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Setup database (MySQL)
4. Run main file
